# Tema Escolhido
Este projeto é uma Single Page Application (SPA) desenvolvida como parte de uma atividade prática de Desenvolvimento Web. O sistema permite a listagem, visualização de detalhes e gerenciamento de favoritos de uma coleção de jogos.

**Catálogo de Jogos Favoritos.** O projeto foca em organizar títulos de jogos, exibindo categorias, descrições e permitindo uma navegação dinâmica entre a lista principal e os detalhes de cada item.

## Tecnologias Utilizadas
* **React:** Framework principal para construção da interface.
* **Vite:** Ferramenta de build rápida para o desenvolvimento.
* **React Router Dom:** Gerenciamento de rotas e navegação na SPA.
* **React Bootstrap:** Framework de CSS para componentes responsivos e estilização.
* **Hooks (useState/useEffect):** Gerenciamento de estado local e efeitos colaterais.

## Pré-requisitos
Antes de começar, você precisará ter instalado em sua máquina:
* [Node.js](https://nodejs.org/)
* Gerenciador de pacotes (npm ou yarn).
